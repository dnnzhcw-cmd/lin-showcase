# 移除span元素下划线的修改方案

## 问题分析

用户希望删除向下滚动提示文字的下划线。当前下划线是通过内联样式中的 `borderBottom` 属性设置的：

```html
<span className="text-xs md:text-sm tracking-wide font-['Space_Grotesk'] pb-0.5 transition-colors group-hover:border-opacity-100" style={{
  color: 'var(--hero-accent-orange)',
  borderBottom: '2px solid var(--hero-accent-orange)',
  borderColor: 'var(--hero-accent-orange)'
}}>
  向下滚动查看完整案例
</span>
```

## 解决方案

### 方案1：直接移除borderBottom样式

```html
<span className="text-xs md:text-sm tracking-wide font-['Space_Grotesk'] pb-0.5 transition-colors" style={{
  color: 'var(--hero-accent-orange)'
}}>
  向下滚动查看完整案例
</span>
```

### 方案2：保留hover效果但移除默认下划线

```html
<span className="text-xs md:text-sm tracking-wide font-['Space_Grotesk'] pb-0.5 transition-colors group-hover:border-b-2 group-hover:border-orange-500" style={{
  color: 'var(--hero-accent-orange)'
}}>
  向下滚动查看完整案例
</span>
```

## 推荐方案

**方案1** - 直接移除下划线，保持简洁的文字效果。这样可以让文字更加干净，不影响整体视觉效果。

## 实施步骤

1. 定位到 `HeroSection.tsx` 文件中的滚动提示文字 `span` 元素
2. 移除 `style` 属性中的 `borderBottom` 和 `borderColor` 样式
3. 移除 `group-hover:border-opacity-100` 类名（因为不再需要）
4. 保存修改并验证效果

## 预期效果

- ✅ 文字下划线被移除
- ✅ 文字颜色保持不变（橙色）
- ✅ 文字大小和字体保持不变
- ✅ 整体视觉效果更加简洁
