## 步骤二：分析项目结构并集成 Everything Claude Code

### 项目结构分析

Everything Claude Code 项目包含以下核心模块：

1. **agents/** - 13个专业子代理
   - architect.md（架构师）
   - code-reviewer.md（代码审查员）
   - security-reviewer.md（安全审查员）
   - planner.md（规划师）
   - 等13个专业代理

2. **commands/** - 24个斜杠命令
   - code-review.md（代码审查）
   - test-coverage.md（测试覆盖率）
   - plan.md（规划）
   - refactor-clean.md（重构清理）
   - 等24个实用命令

3. **rules/** - 8个规则集
   - agents.md（代理规则）
   - coding-style.md（编码风格）
   - security.md（安全规则）
   - testing.md（测试规则）
   - 等8个规则文件

4. **skills/** - 28个专业技能
   - backend-patterns/（后端模式）
   - frontend-patterns/（前端模式）
   - security-review/（安全审查）
   - tdd-workflow/（TDD工作流）
   - 等28个技能包

5. **hooks/** - 自动化钩子
   - hooks.json（钩子配置）
   - scripts/hooks/（钩子脚本）
