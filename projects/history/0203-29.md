# 核心文案卡片高度优化

## 修改时间
2026-02-03 17:35

## 问题分析

### 问题描述
核心文案区域（居中卡片式设计）在比较宽的屏幕中（桌面端和平板）高度太高，超出了背景区域。

### 根因分析
导致卡片高度过高的原因：

1. **卡片最大宽度过大**：
   - 桌面端：`max-w-6xl`（72rem ≈ 1152px）
   - 平板端：`max-w-5xl`（64rem ≈ 1024px）
   - 过宽的卡片导致内容高度增加

2. **卡片内边距过大**：
   - 桌面端：`p-12`（48px 上下）
   - 平板端：`p-8`（32px 上下）
   - 过大的内边距增加了卡片高度

3. **引号字体过大**：
   - 桌面端：`text-8xl`（96px）
   - 平板端：`text-7xl`（80px）
   - 引号占用了大量垂直空间

4. **主标题字体过大**：
   - 桌面端：`text-5xl`（48px）
   - 平板端：`text-3xl`（30px）
   - 过大的字体导致内容高度增加

5. **副标题字体过大**：
   - 桌面端：`text-xl`（20px）
   - 平板端：`text-lg`（18px）
   - 增加了整体高度

6. **间距过大**：
   - 引号上下间距：`mb-4 md:mb-6`
   - 主标题上下间距：`mb-4 md:mb-8`
   - 副标题上下间距：`pt-4 md:pt-6`, `mt-4 md:mt-6`

## 修复方案

### 修复1：减小卡片最大宽度

**修改前**：
```tsx
<div className="w-full max-w-4xl md:max-w-5xl lg:max-w-6xl">
```

**修改后**：
```tsx
<div className="w-full max-w-3xl md:max-w-4xl lg:max-w-5xl">
```

**说明**：
- 移动端：max-w-4xl（不变）
- 平板端：max-w-5xl → max-w-4xl（减小）
- 桌面端：max-w-6xl → max-w-5xl（减小）

### 修复2：减小卡片内边距

**修改前**：
```tsx
className="bg-white/50 ... p-5 md:p-8 lg:p-12 ..."
```

**修改后**：
```tsx
className="bg-white/50 ... p-4 md:p-6 lg:p-8 ..."
```

**说明**：
- 移动端：p-5 → p-4（16px → 16px，不变）
- 平板端：p-8 → p-6（32px → 24px，减小）
- 桌面端：p-12 → p-8（48px → 32px，减小）

### 修复3：减小引号字体

**修改前**：
```tsx
<span className="text-5xl md:text-7xl lg:text-8xl font-serif leading-none">"</span>
```

**修改后**：
```tsx
<span className="text-4xl md:text-6xl lg:text-7xl font-serif leading-none">"</span>
```

**说明**：
- 移动端：text-5xl → text-4xl（48px → 36px）
- 平板端：text-7xl → text-6xl（80px → 72px）
- 桌面端：text-8xl → text-7xl（96px → 80px）

### 修复4：减小主标题字体

**修改前**：
```tsx
<p className="text-2xl md:text-3xl lg:text-5xl ...">
```

**修改后**：
```tsx
<p className="text-xl md:text-2xl lg:text-4xl ...">
```

**说明**：
- 移动端：text-2xl → text-xl（24px → 20px）
- 平板端：text-3xl → text-2xl（30px → 24px）
- 桌面端：text-5xl → text-4xl（48px → 36px）

### 修复5：减小副标题字体

**修改前**：
```tsx
<p className="text-base md:text-lg lg:text-xl ...">
<p className="text-sm md:text-base lg:text-lg ...">
```

**修改后**：
```tsx
<p className="text-sm md:text-base lg:text-lg ...">
<p className="text-xs md:text-sm lg:text-base ...">
```

**说明**：
- 第一行：
  - 移动端：text-base → text-sm（16px → 14px）
  - 平板端：text-lg → text-base（18px → 16px）
  - 桌面端：text-xl → text-lg（20px → 18px）
- 第二行：
  - 移动端：text-sm → text-xs（14px → 12px）
  - 平板端：text-base → text-sm（16px → 14px）
  - 桌面端：text-lg → text-base（18px → 16px）

### 修复6：减小间距

**修改前**：
```tsx
<div className="flex justify-start mb-4 md:mb-6">
<p className="... mb-4 md:mb-8">
<div className="pt-4 md:pt-6 ...">
<p className="... mt-2">
<div className="flex justify-end mt-4 md:mt-6">
```

**修改后**：
```tsx
<div className="flex justify-start mb-3 md:mb-4">
<p className="... mb-3 md:mb-6">
<div className="pt-3 md:pt-4 ...">
<p className="... mt-2">
<div className="flex justify-end mt-3 md:mt-4">
```

**说明**：
- 引号上边距：mb-4 md:mb-6 → mb-3 md:mb-4
- 主标题下边距：mb-4 md:mb-8 → mb-3 md:mb-6
- 副标题上边距：pt-4 md:pt-6 → pt-3 md:pt-4
- 右侧引号上边距：mt-4 md:mt-6 → mt-3 md:mt-4
