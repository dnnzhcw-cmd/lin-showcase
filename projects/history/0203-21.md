## 布局层次分析（从下到上）

### 1. Section 根容器
```tsx
<section id="hero" className="relative min-h-screen flex flex-col overflow-hidden">
```
- **定位**: relative
- **z-index**: auto（默认）
- **背景**: 无（透明）

### 2. 顶部标题区域
```tsx
<div className="h-[120px] md:h-[140px] lg:h-[160px] bg-white border-b border-gray-100 flex flex-col justify-center">
```
- **定位**: static（默认）
- **z-index**: auto
- **背景颜色**: `bg-white`（白色 #ffffff）
- **边框**: `border-b border-gray-100`
- **高度**: 120px / 140px / 160px（响应式）

**字体颜色**:
- 小标题 "探索AI：学习、实践、展示": `var(--text-muted)` (#9ca3af)
- 主标题 "阿林的AI编程之旅": `var(--text-primary)` (#0f172a)

### 3. 背景区域
```tsx
<div className="flex-1 relative bg-gradient-to-br from-yellow-50 to-orange-50 overflow-hidden hero-background">
```
- **定位**: relative
- **z-index**: auto
- **背景颜色**: `from-yellow-50` (#fefce8) → `to-orange-50` (#ffedd5)
- **CSS 伪元素**:
  - `::before`: 渐变网格图案（三个径向渐变）
  - `::after`: 噪点纹理（SVG 滤镜）

### 4. 浮动元素容器
```tsx
<div className="absolute inset-0 overflow-hidden">
  {floatingElements.map((element, index) => (
    <motion.div key={index} className={`absolute ${element.className}`} ...>
  ))}
</div>
```
- **定位**: absolute inset-0
- **z-index**: auto
- **透明度**: 0.1-0.15（循环动画）

**浮动元素颜色**:
- Code 图标: 背景色 `var(--hero-note-yellow)` (#fef08a), 图标色 `var(--hero-accent-primary)` (#ea580c)
- Lightbulb 图标: 背景色 `var(--hero-note-yellow)` (#fef08a), 图标色 `var(--hero-accent-primary)` (#ea580c)
- FileText 图标: 背景色 `var(--hero-note-green)` (#86efac), 图标色 `var(--hero-accent-secondary)` (#16a34a)
- Layers 图标: 背景色 `var(--hero-note-blue)` (#bfdbfe), 图标色 `var(--hero-accent-blue)` (#3b82f6)

### 5. 核心文案容器
```tsx
<div className="absolute inset-0 flex items-center justify-center">
```
- **定位**: absolute inset-0
- **z-index**: auto
- **背景**: 无（透明）

### 6. 核心文案卡片
```tsx
<motion.div className="bg-white/80 backdrop-blur-sm rounded-3xl shadow-2xl p-8 md:p-12 lg:p-16 border border-white/50 relative overflow-hidden cursor-default">
```

- **定位**: static（默认）
- **z-index**: auto
- **背景颜色**: `bg-white/80`（白色 80% 透明度）
- **边框**: `border border-white/50`
- **毛玻璃**: `backdrop-blur-sm`

**装饰线条**:
- 顶部线条: 渐变（透明 → 橙色 #ea580c → 透明），50% 透明度
- 底部线条: 渐变（透明 → 橙色 #ea580c → 透明），50% 透明度

**字体颜色**:
- 引号: `var(--text-quote)` (#e5e7eb)
- 主标题 "总忘长期任务？不是你记性差，是工具太笨重。": `var(--text-primary)` (#0f172a)
- 副标题 "这是我用 AI 辅助开发的第一个产品": `var(--text-secondary)` (#475569)

- **定位**: absolute bottom-12 left-1/2 -translate-x-1/2
- **z-index**: auto
- **背景**: 无（透明）
