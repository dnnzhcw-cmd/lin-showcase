# HeroSection 尺寸适配优化

## 修复时间
2026-02-03 16:50

## 修复目标
修复 HeroSection 的尺寸适配问题，避免顶部标题区域和核心文案卡片重叠，并确保在不同设备上都有良好的显示效果。

## 问题分析

### 原始问题
1. **尺寸重叠**：顶部标题区域和核心文案卡片区域有重叠
2. **固定高度**：使用固定像素高度（120px/140px/160px），在不同设备上比例不一致
3. **响应式不足**：没有充分考虑不同屏幕尺寸的适配需求

### 常见屏幕尺寸分析
- **手机（< 640px）**：
  - iPhone SE: 375x667
  - iPhone 12/13: 390x844
  - iPhone 14 Pro Max: 430x932
- **平板（640px - 1024px）**：
  - iPad Mini: 768x1024
  - iPad Air: 820x1180
  - iPad Pro: 1024x1366
- **电脑（> 1024px）**：
  - 笔记本: 1366x768, 1440x900
  - 台式机: 1920x1080, 2560x1440

## 解决方案

### 方案概述
使用视口单位（vh, vw）替代固定像素，实现按屏幕高度/宽度计算的自适应布局。

### 修复1：外层容器尺寸调整

**原始代码**：
```tsx
<section id="hero" className="relative min-h-screen flex flex-col overflow-hidden">
```

**修复后**：
```tsx
<section id="hero" className="relative h-screen w-screen flex flex-col overflow-hidden">
```

**改进点**：
- `min-h-screen` → `h-screen`：确保占满整个视口高度
- 添加 `w-screen`：确保占满整个视口宽度

### 修复2：顶部标题区域尺寸调整

**原始代码**：
```tsx
<div className="h-[120px] md:h-[140px] lg:h-[160px] bg-white flex flex-col justify-center relative">
```

**修复后**：
```tsx
<div className="h-[14vh] md:h-[15vh] lg:h-[16vh] bg-white flex flex-col justify-center relative">
```

**改进点**：
- 使用视口高度百分比：14vh（手机）/ 15vh（平板）/ 16vh（电脑）
- 字体大小调整：
  - 导航标题：`text-xs md:text-sm lg:text-base`
  - 主标题：`text-3xl md:text-5xl lg:text-7xl`

### 修复3：背景区域尺寸调整

**原始代码**：
```tsx
<div className="flex-1 relative bg-gradient-to-br from-yellow-50 to-orange-50 overflow-hidden hero-background">
```

**修复后**：
```tsx
<div className="h-[86vh] md:h-[85vh] lg:h-[84vh] relative bg-gradient-to-br from-yellow-50 to-orange-50 overflow-hidden hero-background">
```

**改进点**：
- `flex-1` → 精确计算的视口高度：86vh/85vh/84vh
- 确保剩余空间正好填满视口

### 修复4：核心文案卡片尺寸调整

**原始代码**：
```tsx
<div className="absolute inset-0 flex items-center justify-center" style={{ zIndex: 10 }}>
  <div className="max-w-3xl mx-auto px-4">
    <motion.div className="bg-white/50 backdrop-blur-md rounded-3xl shadow-2xl p-6 md:p-10 lg:p-12 border border-white/60 ...">
```

**修复后**：
```tsx
<div className="absolute inset-0 flex items-center justify-center px-4 md:px-6 lg:px-8" style={{ zIndex: 10 }}>
  <div className="w-full max-w-4xl md:max-w-5xl lg:max-w-6xl">
    <motion.div className="bg-white/50 backdrop-blur-md rounded-3xl shadow-2xl p-5 md:p-8 lg:p-12 border border-white/60 mx-auto" style={{ maxWidth: '90vw' }}>
```

**改进点**：
- 外层容器添加内边距：`px-4 md:px-6 lg:px-8`
- 卡片容器宽度增加：`max-w-3xl` → `max-w-4xl md:max-w-5xl lg:max-w-6xl`
- 卡片最大宽度限制：`style={{ maxWidth: '90vw' }}`
- 卡片内边距调整：`p-5 md:p-8 lg:p-12`
- 卡片内容尺寸调整：
  - 引号：`text-5xl md:text-7xl lg:text-8xl`
  - 主标题：`text-2xl md:text-3xl lg:text-5xl`
  - 副标题：`text-base md:text-lg lg:text-xl`

### 修复5：浮动元素位置调整

**原始代码**：
```tsx
const floatingElements = [
  { left: 10, top: 20, ... },   // Code
  { left: 85, top: 15, ... },   // Lightbulb
  { left: 15, top: 70, ... },   // FileText
  { left: 80, top: 75, ... },   // Layers
];
```

**修复后**：
```tsx
const floatingElements = [
  { left: 5, top: 15, ... },    // Code
  { left: 90, top: 10, ... },   // Lightbulb
  { left: 5, top: 80, ... },    // FileText
  { left: 90, top: 85, ... },   // Layers
];
```

**改进点**：
- 调整位置更靠近角落，避免与卡片重叠
- 图标尺寸响应式：`w-10 h-10 md:w-12 md:h-12`
- 内边距响应式：`p-2 md:p-3`

### 修复6：向下滚动提示尺寸调整

**原始代码**：
```tsx
<div className="absolute bottom-12 left-1/2 -translate-x-1/2 flex flex-col items-center gap-3 ...">
  <span className="text-sm tracking-wide font-['Space_Grotesk'] ...">向下滚动查看完整案例</span>
  <div className="w-6 h-6 ...">
    <ChevronDown className="w-4 h-4" />
  </div>
</div>
```

**修复后**：
```tsx
<div className="absolute bottom-8 md:bottom-12 left-1/2 -translate-x-1/2 flex flex-col items-center gap-2 md:gap-3 ...">
  <span className="text-xs md:text-sm tracking-wide font-['Space_Grotesk'] ...">向下滚动查看完整案例</span>
  <div className="w-5 h-5 md:w-6 md:h-6 ...">
    <ChevronDown className="w-3.5 h-3.5 md:w-4 md:h-4" />
  </div>
</div>
```

**改进点**：
- 底部距离调整：`bottom-8 md:bottom-12`
- 间距调整：`gap-2 md:gap-3`
- 文字大小调整：`text-xs md:text-sm`
- 图标尺寸调整：`w-5 h-5 md:w-6 md:h-6`

### 修复7：底部渐变过渡尺寸调整

**原始代码**：
```tsx
<div className="absolute bottom-0 left-0 right-0 h-24 bg-gradient-to-t from-white to-transparent pointer-events-none" />
```

**修复后**：
```tsx
<div className="absolute bottom-0 left-0 right-0 h-12 md:h-24 bg-gradient-to-t from-white to-transparent pointer-events-none" />
```

**改进点**：
- 高度响应式：`h-12 md:h-24`
