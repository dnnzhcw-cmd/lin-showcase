## 组件详细说明

### 高复杂度组件 (4个)
- **Resizable**: 可调整大小的容器，支持四方向调整
- **Carousel**: 轮播图，基于 Embla Carousel，支持循环播放
- **Field**: 表单字段容器，支持三种布局方向
- **Accordion**: 手风琴折叠面板，支持单选/多选模式

### 中等复杂度组件 (4个)
- **InputOTP**: 一次性密码输入，自动聚焦和跳转
- **InputGroup**: 输入框组合，支持四种对齐方式
- **Item**: 列表项组件，支持 icon/image 媒体类型
- **ToggleGroup**: 切换按钮组，支持单选/多选

### 低复杂度组件 (4个)
- **Collapsible**: 轻量级折叠容器
- **Toggle**: 二态切换按钮
- **Sonner**: 通知提示组件
- **ButtonGroup**: 按钮组容器

## 技术亮点

### 1. Carousel 组件
- 使用 Embla Carousel 实现
- 支持水平/垂直轮播
- 键盘导航支持
- 可通过 `setApi` 获取控制 API

### 2. Field 组件
- 三种布局方向：vertical、horizontal、responsive
- 内置错误处理和验证状态
- 支持嵌套分组
- 响应式布局自动适配

### 3. InputOTP 组件
- 专为验证码设计
- 自动聚焦和跳转
- 分组显示（如 3-4-3 格式）
- 闪烁光标动画

### 4. Resizable 组件
- 四方向调整（四个角）
- 手柄拖拽交互
- 可嵌套使用
- 适合分屏布局

## 使用建议

### 常见组合模式
1. **手风琴 FAQ**：Accordion 用于常见问题展示
2. **验证码输入**：InputOTP 用于一次性密码
3. **工具栏切换**：ToggleGroup 用于多选项组
4. **图片轮播**：Carousel 用于图片展示
5. **表单字段**：Field 用于表单布局和验证

### 性能优化
- Carousel 大量图片时使用懒加载
- Accordion 多项内容时使用 collapsible
- Resizable 避免过度嵌套
- Sonner 限制同时显示的通知数量

### 无障碍建议
- 所有交互组件支持键盘导航
- Accordion 使用 aria-expanded 属性
- Carousel 包含 aria-roledescription
- Field 正确关联 aria-invalid
