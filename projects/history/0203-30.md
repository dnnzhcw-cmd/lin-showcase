# HeroHeader 极简设计优化

## 修改时间
2026-02-03 17:40

## 问题描述

用户反馈 HeroHeader 组件存在以下问题：
1. ❌ 背景颜色为白色，不够协调
2. ❌ 电脑端和平板端字体太大
3. ❌ 上下行距太大，超出了容器范围
4. ❌ 没有实现极简设计
5. ❌ 没有充足的负空间
6. ❌ 字体颜色不够协调

## 修改目标

实现极简设计，提供：
- ✅ 灰蓝色背景，与整体设计协调
- ✅ 合适的字体大小，不会超出容器
- ✅ 适度的行距和间距，充足的负空间
- ✅ 协调的字体颜色
- ✅ 简洁优雅的视觉效果

## 修改内容

### 1. 背景颜色优化

**修改前**：
```tsx
<div className="... bg-white ...">
```

**修改后**：
```tsx
<div className="... bg-slate-50 ...">
```

**说明**：
- 白色（#FFFFFF）→ 灰蓝色（slate-50, #F8FAFC）
- 灰蓝色背景更柔和，与暖色系的背景区域形成对比
- 提升整体视觉协调性

### 2. 导航标题优化

#### 2.1 字体大小

**修改前**：
```tsx
className="text-sm md:text-base lg:text-lg ..."
```

**修改后**：
```tsx
className="text-xs md:text-sm lg:text-base ..."
```

**说明**：
- 移动端：text-sm → text-xs（14px → 12px）
- 平板端：text-base → text-sm（16px → 14px）
- 桌面端：text-lg → text-base（18px → 16px）
- 减小字体大小，增加负空间

#### 2.2 字间距

**修改前**：
```tsx
className="... tracking-widest ..."
```

**修改后**：
```tsx
className="... tracking-wide ..."
```

**说明**：
- tracking-widest（0.2em）→ tracking-wide（0.1em）
- 减小字间距，提升可读性

#### 2.3 字体粗细

**修改前**：
```tsx
className="... font-['Space_Grotesk']"
```

**修改后**：
```tsx
className="... font-['Space_Grotesk'] font-medium"
```

**说明**：
- 添加 font-medium，字体更有质感
- 字重 500，介于 normal 和 bold 之间

#### 2.4 下边距

**修改前**：
```tsx
className="... mb-2 md:mb-3 ..."
```

**修改后**：
```tsx
className="... mb-1 md:mb-2 ..."
```

**说明**：
- 移动端：mb-2 → mb-1（8px → 4px）
- 平板端：mb-3 → mb-2（12px → 8px）
- 减小下边距，增加负空间

#### 2.5 字体颜色

**修改前**：
```tsx
style={{ color: 'var(--text-muted)' }}
```

**修改后**：
```tsx
style={{ color: '#64748b' }}
```

**说明**：
- 使用 Slate-500（#64748b），与灰蓝色背景协调
- 更柔和的灰色，提升视觉舒适度

### 3. 主标题优化

#### 3.1 字体大小

**修改前**：
```tsx
className="text-3xl md:text-5xl lg:text-7xl ..."
```

**修改后**：
```tsx
className="text-xl md:text-3xl lg:text-4xl ..."
```

**说明**：
- 移动端：text-3xl → text-xl（30px → 20px）
- 平板端：text-5xl → text-3xl（48px → 30px）
- 桌面端：text-7xl → text-4xl（72px → 36px）
- 显著减小字体大小，不会超出容器

#### 3.2 字体粗细

**修改前**：
```tsx
className="... font-bold ..."
```

**修改后**：
```tsx
className="... font-semibold ..."
```

**说明**：
- font-bold（700）→ font-semibold（600）
- 减轻字体重量，更加优雅

#### 3.3 行高

**修改前**：
```tsx
className="... leading-tight"
```

**修改后**：
```tsx
className="... leading-tight"
```

**说明**：
- 保持 leading-tight，确保紧凑的行高

#### 3.4 字体颜色

**修改前**：
```tsx
style={{ color: 'var(--text-primary)' }}
```

**修改后**：
```tsx
style={{ color: '#1e293b' }}
```

**说明**：
- 使用 Slate-800（#1e293b），比原 #0f172a 更柔和
- 与灰蓝色背景和导航标题颜色协调

5. **优雅排版**：紧凑但不拥挤

## 极简设计原则应用

1. **少即是多**：去除不必要的装饰和过大元素
2. **负空间**：充分利用空白，提升视觉舒适度
3. **简洁性**：字体、颜色、间距都保持简洁
4. **一致性**：整体风格统一协调
5. **功能性**：每个元素都有明确的作用
