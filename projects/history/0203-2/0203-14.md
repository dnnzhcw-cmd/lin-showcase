#### 3.2 分析 SKILL.md 设计指南

**关键要点**：

1. **设计思维**
   - 选择大胆的美学方向（杂志/编辑风 + 极简主义）
   - 避免常见 AI 套路（Inter 字体、紫色渐变、陈词滥调的配色）

2. **前端美学指南**
   - **排版**：避免 Inter、Roboto 等通用字体，使用独特的展示字体与精致的正文字体配对
   - **色彩与主题**：避免白色背景上的紫色渐变，使用强烈的点缀色配合主导色彩
   - **动效**：使用一次精心编排的页面加载配合交错揭示
   - **空间构成**：不对称布局、对角线流动、充足的负空间
   - **背景与视觉细节**：渐变网格、噪点纹理、几何图案、分层透明度

3. **AI 产品展示页设计模式**
   - Hero 区域：简洁有力的主标题、清晰的价值主张、强烈的 CTA 按钮、视觉化产品演示
   - 文案写作：避免技术术语堆砌，使用场景化的描述

4. **互动与动态元素增强**
   - 虚拟便签组件：使用柔和自然颜色、手写字体或纸质质感
   - 状态切换交互模式：Tab 切换、折叠面板、模态框、侧边栏抽屉
   - 微交互设计：按钮悬停效果、链接下划线动画、表单交互、滚动触发效果

```css
:root {
  /* 主色调 - 暖色系 */

  --primary-bg-end: #ffedd5;    /* 浅橙 orange-50 */
  
  /* 强调色 */
  --accent-primary: #ea580c;    /* 暖橙 orange-600 */
  --accent-secondary: #16a34a;  /* 绿色 green-600 */
  
  /* 文字颜色 - 高对比度 */
  --text-primary: #0f172a;      /* 深蓝 gray-900 */
  --text-secondary: #475569;    /* 中灰 gray-600 */
}
```
