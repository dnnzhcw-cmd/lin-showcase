# HeroHeader 组件拆分

## 拆分时间
2026-02-03 17:05

## 拆分目标
将 HeroSection 中的顶部标题区域（"探索AI：学习、实践、展示" 和 "阿林的AI编程之旅"）单独拆分为一个独立的组件，提高代码的可维护性和复用性。

## 拆分内容

### 新建 HeroHeader 组件

**文件**：`src/components/sections/HeroHeader.tsx`

```tsx
'use client';

import { motion } from 'framer-motion';

export default function HeroHeader() {
  return (
    <div className="h-[14vh] md:h-[15vh] lg:h-[16vh] bg-white flex flex-col justify-center relative overflow-hidden">
      <div className="max-w-7xl mx-auto px-4">
        <motion.p
          initial={{ opacity: 0, y: -20 }}
          animate={{ opacity: 1, y: 0 }}
          transition={{ duration: 0.6, delay: 0 }}
          className="text-sm md:text-base lg:text-lg mb-2 md:mb-3 tracking-widest uppercase text-center font-['Space_Grotesk']"
          style={{ color: 'var(--text-muted)' }}
        >
          探索AI：学习、实践、展示
        </motion.p>
        <motion.h1
          initial={{ opacity: 0, y: -20 }}
          animate={{ opacity: 1, y: 0 }}
          transition={{ duration: 0.6, delay: 0 }}
          className="text-3xl md:text-5xl lg:text-7xl font-bold text-center font-['Outfit'] leading-tight"
          style={{ color: 'var(--text-primary)' }}
        >
          阿林的AI编程之旅
        </motion.h1>
      </div>

      {/* 底部阴影分隔 */}
      <div className="absolute bottom-0 left-0 right-0 h-4 bg-gradient-to-b from-gray-100 to-transparent" />
      <div className="absolute bottom-0 left-0 right-0 h-[1px] bg-gradient-to-r from-transparent via-gray-200 to-transparent" />
    </div>
  );
}
```

### 修改 HeroSection 组件

**修改内容**：

1. **导入 HeroHeader 组件**：
```tsx
import HeroHeader from './HeroHeader';
```

2. **使用 HeroHeader 组件替换原来的顶部标题区域**：
```tsx
return (
  <section id="hero" className="relative h-screen w-screen flex flex-col overflow-hidden">
    {/* 顶部标题区域 */}
    <HeroHeader />

    {/* 背景区域 */}
    <div className="h-[86vh] md:h-[85vh] lg:h-[84vh] ...">
      {/* 背景区域内容 */}
    </div>
  </section>
);
```

## 拆分优势

### 1. 代码可维护性
- ✅ HeroHeader 组件独立管理顶部标题区域的逻辑和样式
- ✅ HeroSection 组件专注于背景区域和核心内容
- ✅ 职责清晰，代码更易理解

### 2. 代码复用性
- ✅ HeroHeader 组件可以在其他页面或组件中复用
- ✅ 便于在多个地方使用相同的头部设计

### 3. 测试便利性
- ✅ HeroHeader 组件可以单独测试
- ✅ HeroSection 组件的测试更简单

### 4. 开发效率
- ✅ 团队成员可以并行开发不同组件
- ✅ 修改头部样式时不会影响背景区域
