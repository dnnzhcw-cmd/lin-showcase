# HeroSection 视觉优化修复

## 修复时间
2026-02-03 16:45

## 修复目标
根据用户反馈修复 HeroSection 的三个视觉问题：
1. 顶部标题区域和核心文案卡片没有分隔
2. 4 个浮动元素的饱和度非常低，非常透明
3. 白色毛玻璃卡片需要能看到浅黄背景

## 修复内容

### 修复1：添加顶部标题区域和核心文案卡片的分隔

**问题**：顶部标题区域（白色背景）和核心文案卡片（浅黄背景）之间没有明显的视觉分隔

**解决方案**：
在顶部标题区域底部添加两层分隔效果：
1. **阴影渐变**：`h-4 bg-gradient-to-b from-gray-100 to-transparent`
2. **线条分隔**：`h-[1px] bg-gradient-to-r from-transparent via-gray-200 to-transparent`

```tsx
<div className="h-[120px] md:h-[140px] lg:h-[160px] bg-white flex flex-col justify-center relative">
  <div className="max-w-7xl mx-auto px-4">
    {/* 标题内容 */}
  </div>
  {/* 底部阴影分隔 */}
  <div className="absolute bottom-0 left-0 right-0 h-4 bg-gradient-to-b from-gray-100 to-transparent" />
  <div className="absolute bottom-0 left-0 right-0 h-[1px] bg-gradient-to-r from-transparent via-gray-200 to-transparent" />
</div>
```

### 修复2：提高浮动元素的饱和度和透明度

**问题**：浮动元素的饱和度非常低，非常透明（opacity: 0.1-0.15）

**解决方案**：
1. **提高透明度**：将 opacity 从 0.1-0.15 提高到 0.25-0.35
2. **优化颜色使用**：
   - Code 和 Lightbulb 使用 `--hero-accent-orange` (橙色)
   - FileText 使用 `--hero-accent-green` (绿色)
   - Layers 使用 `--hero-accent-blue` (蓝色)

```tsx
const floatingElements = [
  {
    icon: <Code className="w-12 h-12" />,
    style: {
      backgroundColor: 'var(--hero-note-yellow)',
      color: 'var(--hero-accent-orange)', // 使用橙色
    },
    opacity: 0.3, // 从 0.12 提高
    // ...
  },
  {
    icon: <Lightbulb className="w-10 h-10" />,
    style: {
      backgroundColor: 'var(--hero-note-yellow)',
      color: 'var(--hero-accent-orange)', // 使用橙色
    },
    opacity: 0.35, // 从 0.15 提高
    // ...
  },
  {
    icon: <FileText className="w-11 h-11" />,
    style: {
      backgroundColor: 'var(--hero-note-green)',
      color: 'var(--hero-accent-green)', // 使用绿色
    },
    opacity: 0.25, // 从 0.1 提高
    // ...
  },
  {
    icon: <Layers className="w-10 h-10" />,
    style: {
      backgroundColor: 'var(--hero-note-blue)',
      color: 'var(--hero-accent-blue)', // 使用蓝色
    },
    opacity: 0.3, // 从 0.12 提高
    // ...
  },
];
```

### 修复3：调整白色毛玻璃卡片透明度

**问题**：卡片背景太不透明（bg-white/80），看不到浅黄背景

**解决方案**：
1. **降低背景不透明度**：`bg-white/80` → `bg-white/50`
2. **增强毛玻璃效果**：`backdrop-blur-sm` → `backdrop-blur-md`
3. **调整边框透明度**：`border-white/50` → `border-white/60`

```tsx
<motion.div
  className="bg-white/50 backdrop-blur-md rounded-3xl shadow-2xl p-6 md:p-10 lg:p-12 border border-white/60 relative overflow-hidden cursor-default"
>
  {/* 卡片内容 */}
</motion.div>
```
