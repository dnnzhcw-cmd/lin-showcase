## 🧩 核心 UI 组件说明

### 1. NoteCard (便签卡片)
**用途**: 展示任务/痛点的可交互便签
**Props**:
- `title`: 标题
- `description`: 描述
- `status`: 状态 ('not-started' | 'in-progress' | 'completed')
- `onStatusChange`: 状态切换回调

**交互**:
- 点击卡片循环切换状态
- 状态变化时有颜色动画

**视觉**:
- 未开始: 灰色边框
- 进行中: 黄色边框 + 摇晃动画
- 已完成: 绿色边框 + 勾选图标

---

### 2. Accordion (手风琴)
**用途**: 折叠/展开内容区域
**Props**:
- `title`: 标题
- `children`: 内容
- `isOpen`: 是否展开

**交互**:
- 点击标题切换展开/收起
- 展开/收起有平滑动画

---

### 3. Timeline (时间线)
**用途**: 展示学习路径或技术栈演进
**结构**:
- 垂直线连接
- 时间节点图标
- 左右交替布局 (桌面端)
- 单列布局 (移动端)

---

### 4. ComparisonTable (对比表格)
**用途**: 竞品功能对比
**结构**:
- 表头: 功能点 + 多个产品列
- 表体: 星级评分
- 响应式: 移动端横向滚动

---

### 5. SocialLinks (社交媒体链接)
**用途**: 外部链接展示
**交互**: 
- Hover 时不透明度变化
- 点击在新标签页打开

---

## 🎭 动画交互设计

### Framer Motion 使用场景

| 组件 | 动画类型 | 效果 |
|------|---------|------|
| HeroSection 浮动便签 | 持续动画 | 透明度 + 位移 + 旋转 |
| Section 滚动进入 | Scroll Trigger | 淡入 + 向上位移 |
| NoteCard 状态切换 | 过渡动画 | 边框颜色变化 + 缩放 |
| Accordion 展开/收起 | 布局动画 | 高度变化 + 内容淡入淡出 |
| NavItem Hover | 微交互 | 缩放 + 颜色变化 |
| ScrollTopButton | 显隐动画 | 缩放 + 透明度 |

---

## 📐 响应式断点

| 断点 | 屏幕宽度 | 布局调整 |
|------|---------|---------|
| mobile | < 640px | 单列布局, 汉堡菜单 |
| tablet | 640px - 1024px | 部分双列, 保持顶部导航 |
| desktop | 1024px - 1280px | 多列布局, 左侧导航 |
| large-desktop | > 1280px | 宽屏优化, 最大宽度限制 |

```css
:root {
  /* 主色调 */
  --color-primary: #111827;     /* 深灰 - 主要文字 */

  --bg-primary: #ffffff;        /* 白色 */
  --bg-secondary: #f9fafb;      /* 浅灰 */
  --bg-hero: #111827;           /* 深灰渐变 */

  --hover-bg: rgba(0,0,0,0.05); /* 悬停背景 */
  --active-bg: rgba(0,0,0,0.1); /* 点击背景 */
}

```

```
  ↓
Hero Section (自动滚动提示)
  ↓
[左侧导航栏] ← 实时高亮当前章节
  ├─ 首页 → Hero
  ├─ 项目路演 → PainPoints / MVPDemo / Competitor
  ├─ 学习复盘 → LearningJourney / TechStack
  ├─ 联系我 → CallToAction

[滚动监听]
  ↓
更新 activeSection 状态
  ↓
更新导航栏高亮状态
```
